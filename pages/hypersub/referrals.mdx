# Referrals

Reward subscribers who share your Hypersub with referral links. Subscribers that share links leading to successful conversions are compensated with a percentage of that subscription's revenue.

To get started, go to your dashboard and select the referrals tab.

### Create

Create a referral by using the button at the top left of the page. Enter a title for your referral and select a percentage. Confirm to open your wallet and sign the transaction. Next you will need to assign the link and share with subscribers.

### Assign

Assign referral links to subscribers. Use the more icon to the right of the referral's row in the table. Select create link from the menu. Enter the subscriber's address and use the create button to generate the link. Copy the link and share it with the subscriber.

### Edit

Referrals can be edited at any time. Use the more icon to the right of the referral's row in the table. Select edit from the menu. Change the title, percentage and other options covered below. Confirm and sign.

Delete a referral by selecting the option from the menu and confirming.

### Default Link

All subscribers are assigned a referral link. Set a link to default by switching it on. Immediately after time is purchased subscribers are shown the default referral link.

Subscribers can always access the default link from the earn tab on your subscription page.

### Single Address

Create a referral link that is only for one address. Set your link to single address by switching it on and entering an address. Single address referrals are automatially assigned to the entered address.

Keep track of who has what referral. Allows bespoke referrals for creators with more advanced referral structures.

### Permanet Link

Referral links can be made permanent, meaning they can never be edited or deleted. Set your link to permenent by switching it on. This setting has a strong effect and gives a subscriber the ability make referrals indefinitely so long as they have the link.
