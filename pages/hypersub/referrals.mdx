Reward members who share your Hypersub with referral links. Members who share links that successfully convert earn a percentage of the purchase revenue.

To get started, go to your Dashboard and click the Referrals tab.

### Create Referral

1. Click the Create button, enter a title and select a percentage
2. Review, confirm and sign the transaction

### Create Link

Create and share referral links with members.

1. Click the ••• icon in the referral's table row, then click Create Link in the menu
2. Enter the member's account and Click the Create button to generate the link
3. Copy the link and share it with the member

### Edit 

Referrals can be edited at any time.

1. Click the more icon in the referral's table row, then click Edit in the menu.
2. Change the title, percentage or other advanced options
3. Confirm and sign the transaction

Delete a referral by clicking Delete in the menu and confirming.

## Advanced

Access advanced referral options by clicking the Advanced button.

### Default Link

Share a default referral link with each new member that joins your membership. Set a link to default by switching it on.

The default link displays each time a member joins, adds time, our uses the share link on your membership profile.

There can only be one default link. Each new referral is automatically set to default, and will override any other link's defualt setting.

### Permanent Link

Referral links can be made permanent, meaning they can never be edited or deleted. Set your link to permenent by switching it on.

This setting has a strong effect and allows a member to make referrals indefinitely.

### Single Account

Create a referral link that is only for one member. Set your link to single account by switching it on, and entering an account.

Keep track of who has what referral. Allows unique referral links for creators with more advanced affiliate structures.
